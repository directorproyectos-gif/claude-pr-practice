# CLAUDE

Repositorio de práctica para aprender el flujo de Pull Requests con Claude Code.

## Qué es esto

Un espacio de pruebas para experimentar con git, GitHub y PRs de forma segura.

## Cómo contribuir

1. Crea una rama a partir de `main`.
2. Haz tus cambios y confírmalos con un mensaje claro.
3. Abre un Pull Request describiendo qué cambia y por qué.
